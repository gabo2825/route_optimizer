# Simulador Logístico: Curva de Hilbert vs. Orden X (Versión Unificada)

Este proyecto es un prototipo interactivo desarrollado para una monografía académica de Matemática Aplicada. Compara la eficiencia de rutas logísticas utilizando el ordenamiento fractal (Curva de Hilbert) frente a un enfoque tradicional (ordenamiento por coordenada X), calculando distancias Manhattan en una cuadrícula urbana de $8 \times 8$.

## Ejecución del Proyecto

Esta versión ha sido simplificada y corre **100% del lado del cliente** (en el navegador web). No requiere instalar Python, ni servidores web, ni base de datos. Es totalmente portable.

Para abrir la aplicación interactiva:

1.  Ve al directorio raíz de este proyecto (`/Users/gabo/.gemini/antigravity/scratch/hilbert_logistics`).
2.  Busca el archivo **`index.html`**.
3.  Dale **doble clic** para abrirlo en tu navegador favorito (Chrome, Safari, Edge, Firefox).
4.  ¡Listo! Puedes usar el control deslizante (slider) y el botón "Generar Rutas" inmediatamente.

## Estructura de Archivos

*   `index.html`: Contiene toda la estructura visual, estilos de diseño premium y la lógica matemática de la curva de Hilbert escrita en JavaScript.
*   `EXPLICACION.md`: Guion y material de soporte teórico para tu defensa de la monografía.
